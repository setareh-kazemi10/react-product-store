# 🛍️ React Product Store

A simple React application that fetches products from the Fake Store API and displays them using Context API.

---

## 🚀 Features

- Fetch products from Fake Store API
- Display product image
- Product title
- Product description
- Product price
- Product category
- Product rating
- Product count
- Global state using Context API
- Responsive layout with Bootstrap

---

## 🛠️ Technologies

- React
- Context API
- Fetch API
- Bootstrap 5
- Fake Store API

---

## 📦 Installation

```bash
git clone https://github.com/setareh-kazemi10/react-product-store.git
```

```bash
cd react-product-store
```

```bash
npm install
```

```bash
npm run dev
```
